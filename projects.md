Here are some of the key projects I have worked on.

## [Tree Visualizer](https://github.com/anikdc/Tree-Visualiser)
An application to visualize trees dynamically.
- Uses Tkinter and MatPlotLib in Python with web deployment using PyScript.
- Learnt how to work with trees data structure, visualize it, and deploy code on the web.
- Run it on your browser [here](/treeviz.html).

## [Ray Casting Engine](https://github.com/anikdc/Ray-Casting) *(Link to be updated)*
Made a ray casting engine with a live mini-map.
- Uses Python with Object Oriented Programming Principles and trigonometric equations.
- Converts a 2D array into a 3D first-person traversable space.

## [Comparative Analysis of Image Denoising Methods](https://github.com/anikdc/Image-Denoising) *(Link to be updated)*
A detailed, objective comparison between different modern image denoising methods.
- Uses OpenCV and MatPlotLib libraries.
- Implemented Non-Local Means, Bilateral Filter, Median Filter, and Adaptive Median Filter for noise reduction. 
- Uses PSNR and SSIM to judge performance.
- Conducted multiple paper reviews, translating academic knowledge into testable code.
